Actions

#Data Actions

#create(...mods)

declare function create(...mods: Mod[]): ScriptBot | ScriptBot[];

Creates a new bot or combination of bots with the given mods. Also triggers @onCreate on all the created bots. By default, bots are created with a unique #id, #auxCreator set to the current bot.id, and #space set to "shared".

Each parameter is a mod that should be applied to the new bot(s). If no parameters are specified, then the new bot will have its #auxCreator set to bot.id and #space set to "shared". If an array of mods is used for a parameter, then one bot will be created for each unique combination of mods.

#Examples:

  1. Create an empty bot:
let myNewBot = create();
  1. Create a red bot:
let myRedBot = create({
    auxColor: "red"
});
  1. Create a parent and a child bot:
let myParentBot = create({
    auxCreator: null,
    auxLabel: "Parent"
});
let myChildBot = create({
    auxCreator: getID(myParentBot),
    auxLabel: "Child"
});
  1. Create a empty bot in the "tempLocal" space:
let myBot = create({ space: "tempLocal" });
  1. Create a bot from multiple mods:
// myBot is placed in the "myContext" context and is colored green
let myBot = create({ myContext: true }, {
    auxColor: "green"
});
  1. Create a red bot and a blue bot:
let [myRedBot, myBlueBot] = create({ auxCreator: null}, [
    {
        "auxColor": "red"
    },
    {
        "auxColor": "blue"
    }
]);

#destroy(bot)

declare function destroy(bot: Bot | string | Bot[]): void;

Removes the given bot, list of bots, or bot by #id and triggers @onDestroy for each of the destroyed bots.

Note that only destroyable bots will be destroyed. Bots which have #auxDestroyable set to false will not be destroyed.

Also note that all bots which have #auxCreator set to the #id of a destroyed bot will also be destroyed (unless they are not destroyable). This happens recursively until there are no more bots to destroy.

#Examples:

  1. Destroying a single bot:
// Destroy a single bot that has #name set to "bob"
destroy(getBot("#name", "bob"));
  1. Destroying a list of bots:
// Destroy all the bots that have #auxColor set to "red"
destroy(getBots("#auxColor", "red"));
  1. Destroying a bot by its id
// Destroy the bot with the #id: "config"
destroy("config");

#getBots(..filters)

declare function getBots(...filters: ((bot: Bot) => boolean)[]): Bot[];

Gets an array of bots that match all of the given filter(s). The returned array is sorted alphabetically by the #id tag.

Each parameter is a bot filter. If no filters are specified, then all bots in the channel are returned. If multiple filters are specified, then only the bots that match all of the filters are returned.

#Examples

  1. Find all bots with the "test" tag.
let bots = getBots(byTag("#test"));
  1. Find all bots with #name set to "bob" and in the "people" context.
let bots = getBots(byTag("#name", "bob"), inContext("people"));

#getBots(tag, value?)

declare function getBots(tag: string, filter?: any | TagFilter): Bot[];

Gets an array of bots that match the given tag and value. The returned array is sorted alphabetically by the #id tag.

The first parameter is the name of the tag. Bots that have this tag will be included as long as they also match the second parameter.

The second parameter is the value the tag should match. If not specified, then all bots with the tag will be included. If specified, then only bots that have the same tag and value will be included. If you specify a function as the value, then it will be used to match tag values.

This function behaves exactly like getBots(...filters) with byTag(tag, value?)

#Examples

  1. Find all the bots with #name set to "bob".
let bots = getBots("#name", "bob");
  1. Find all bots with a height larger than 2.
let bots = getBots("#height", height => height > 2);
  1. Find all bots with the "test" tag.
let bots = getBots("#test");

#getBot(...filters)

declare function getBot(...filters: BotFilterFunction[]): Bot;

Get the first bot that matches all of the given filter(s). If multiple bots match the given filter(s), then bots are sorted alphabetically by the #id tag and the first one is returned. If no bots match the given filter(s), then undefined is returned.

Each parameter is a bot filter. If no filters are specified, then the first bot in the channel is returned. If multiple filters are specified, then the first bot that matches all of the filters is returned.

#Examples

  1. Find a bot with the "test" tag.
let foundBot = getBot(byTag("#test"));
  1. Find a bot with #name set to "bob" and in the "people" context.
let foundBot = getBot(byTag("#name", "bob"), inContext("people"));

#getBot(tag, filter?)

declare function getBot(tag: string, filter?: any | TagFilter): Bot;

Gets the first bot that matches the given tag and value. If multiple bots match the given tag and value, then bots are sorted alphabetically by the #id tag and the first one is returned. If no bots match the given tag and value, then undefined is returned.

The first parameter is the name of the tag to search for.

The second parameter is the value the tag should match. If not specified, then the first bot with the tag will be returned. If specified, then the first bot that has the same tag and value will be returned. If you specify a function as the value, then it will be used to match tag values.

This function behaves exactly like getBot(...filters) with byTag(tag, value?)

#Examples

  1. Find the first bot with #name set to "bob".
let foundBot = getBot("#name", "bob");
  1. Find the first bot with a height larger than 2.
let foundBot = getBot("#height", height => height > 2);
  1. Find the first bot with the "test" tag.
let foundBot = getBot("#test");

#getID(bot)

declare function getID(bot: Bot | string): string;

Gets the ID of the given bot.

The first parameter is the bot whose ID should be retrieved. If given a bot ID, then it will be returned. If given null or something that is not a bot, then null will be returned.

#Examples

  1. Get the ID of the current bot.
let id = getID(bot);
  1. Get the ID of a bot with #name set to "bob".
let id = getID(getBot("#name", "bob"));

#getJSON(data)

declare function getJSON(data: any): string;

Gets the JSON representation of the given data.

The first parameter is the data that should be cloned into the JSON format. If given a bot, then the returned JSON will be able to be able to be converted back into a mod via getMod(bot).

#Examples

  1. Store a copy of a bot in a tag.
let bob = getBot("#name", "bob");
tags.savedBot = getJSON(bob);

#getMod(bot)

declare function getMod(bot: any, ...tags: (string | RegExp)[]): Mod;

Gets a mod for the given bot and list of tags. The returned mod will only contain tags that are contained in the given list.

The first parameter is the bot or JSON string that should be cloned into a mod.

The other parameters are the tags that should be included in the returned mod. If no other parameters are specified, then all tags are included. If a parameter is a RegEx (Regular Expression), then it will include all the tags that match the expression.

#Examples

  1. Get a mod with the current bot's tags.
let mod = getMod(bot);
  1. Get a mod with only the #auxColor and #auxLabel tags.
let mod = getMod(bot, "auxColor", "auxLabel");
  1. Get a mod with every tag that starts with "on".
let mod = getMod(bot, /^on/g);

#Bot Filters

#byTag(tag, value?)

declare function byTag(tag: string, filter?: TagFilter): BotFilterFunction;

Creates a bot filter which includes bots that have the given tag that matches the given value.

The first parameter is the name of the tag. Bots that have this tag will be included as long as they also match the second parameter.

The second parameter is the value that the tag should match. If not specified, then all bots with the tag will be included. If specified, then only bots that have the same tag value will be included. If you specify a function as the value, then it will be used to match tag values.

#Examples

  1. Find all the bots with #name set to "bob".
let bots = getBots(byTag("#name", "bob"));
  1. Find all bots with a height larger than 2.
let bots = getBots(byTag("#height", height => height > 2));
  1. Find all bots with the "test" tag.
let bots = getBots(byTag("#test"));

#inContext(context)

declare function inContext(context: string): BotFilterFunction;

Creates a bot filter which includes bots that are in the given context. That is, they have the given tag set to true.

The first parameter is the name of the context.

This function behaves exactly like byTag(tag, value?) with the `value` parameter set to `true`.

#Examples

  1. Find all the bots in the "test" context.
let bots = getBots(inContext("test"));

#atPosition(context, x, y)

Creates a bot filter which includes bots that are in the given context and at the given X and Y position.

When this filter is used with getBots(...filters), the returned bots are sorted in the same order that they are stacked. This means that the first bot in the array is at the bottom of the stack and the last bot is at the top of the stack (assuming they're stackable).

The first parameter is the name of the context.

The second parameter is the X position. That is, the left-right position of the bots in the context.

The third parameter is the Y position. That is, the forward-backward position of the bots in the context.

#Examples

  1. Find all the bots at (1, 2) in the "test" context.
let bots = getBots(atPosition("test", 1, 2));

#inStack(bot, context)

Creates a bot filter which includes bots that are in the same stack as the given bot. The given bot will always be included by this filter as long the given bot is in the given context.

When this filter is used with getBots(...filters), the returned bots are sorted in the same order that they are stacked. This means that the first bot in the array is at the bottom of the stack and the last bot is at the top of the stack (assuming they're stackable).

The first parameter is the bot that other bots should be in the same stack with.

The second parameter is the name of the context.

#Examples

  1. Find all bots in the same stack as this in the "test" context.
let bots = getBots(inStack(this, "test"));

#byCreator(bot)

declare function byCreator(bot: Bot | string): BotFilterFunction;

Creates a bot filter which includes bots that were created by the given bot. That is, they have #auxCreator set to the #id of the given bot.

The first parameter is the bot that created the other bots.

This function behaves exactly like byTag("auxCreator", getID(bot)).

#Examples

  1. Find all the bots created by this bot.
let bots = getBots(byCreator(this));

#bySpace(space)

declare function bySpace(space: string): BotFilterFunction;

Creates a bot filter which includes bots that are in the given space. That is, they have #space set to the given value.

The first parameter is the space that the bots are in.

This function behaves exactly like byTag("space", space).

#Examples

  1. Find all bots in the "tempLocal" space.
let bots = getBots(bySpace("tempLocal"));

#Event Actions

#shout(name, arg?)

declare function shout(name: string, arg?: any): any[];

Sends a shout to all bots that are #auxListening and have a listen tag for the specified name. Optionally includes a custom that argument. Also triggers @onListen and @onAnyListen.

The first parameter is the name of the shout. e.g. Using "onClick" for the name will trigger all @onClick listeners.

The second parameter is the that argument to send with the shout. You do not need to specify this parameter if you do not want to.

#Examples:

  1. Send a @reset event to all bots:
shout("reset");
  1. Send a @hello event with your name:
shout("hello", "Bob");

#whisper(bot, name, arg?)

declare function whisper(bot: (Bot | string)[] | Bot | string, eventName: string, arg?: any): any[];

Sends a whisper to the specified bot(s) that are #auxListening and have a listen tag for the given name. Optionally includes a custom that argument. Also triggers @onListen and @onAnyListen.

The first parameter is the bot, array of bots, bot #id, or array of bot #id that the whisper should be sent to.

The second parameter is the name of the whisper. e.g. Using "onClick" for the name will trigger the @onClick listener for the specified bots.

The third parameter is the that argument to send with the shout. You do not need to specify this parameter if you do not want to.

#Examples:

  1. Send a @reset event to all bots named "Bob":
let bots = getBots("#name", "Bob");
whisper(bots, "reset");
  1. Send a @setColor event to ourself:
whisper(this, "setColor", "red");

#Player Actions

#player.replaceDragBot(botOrMod)

declare function replaceDragBot(bot: Mod): BotAction;

Replaces the bot that the user is dragging with the given bot or mod. Only works if called from inside a @onBotDrag or @onAnyBotDrag listen tag.

The first parameter is the bot or mod that should be dragged. If given a bot, then that bot's @onBotDrag will be skippped but @onBotDrop will be called.

#Examples:

  1. Drag a clone of this bot.
let clone = create(this);
player.replaceDragBot(clone);
  1. Drag a mod that makes other bots red.
player.replaceDragBot({
    "auxColor": "red"
});

#Server Actions

#server.setupChannel(channel, botOrMod?)

declare function setupChannel(channel: string, botOrMod?: Mod): void;

Sends an action to the server which will create the given channel if it does not exist and places a clone of the given bot or mod in it. The new bot will only be added to the channel if the channel did not exist. The action is only executed if @onChannelAction has been configured to perform events in device actions.

The first parameter is the ID of the channel that should be setup.

The second parameter is optional and is the bot or mod that should be cloned into the new channel. After the bot is created, @onCreate will be triggered.

#Examples:

Before server.setupChannel() will work, you need to setup the @onChannelAction tag to execute events from remote devices. This is because when you run server.setupChannel() from a PC or phone (a.k.a. A "remote device"), it needs to send an event to the server to actually create the new channel. By default, AUX does not perform actions which are received from a remote device. You must instruct AUX to execute these yourself.

Ensure that @onChannelAction is setup properly. The following script will execute setup_channel events which were sent from other devices:

// Device actions are actions which were sent to this computer from another device
if (that.action.type === 'device') {
    // The setup_channel action is the action that is created
    // from server.setupChannel()
    if (that.action.event.type === 'setup_channel') {
        // Execute the setup_channel action
        action.perform(that.action.event);
    }
}
  1. Create the "test" channel:
server.setupChannel("test");
  1. Create the "test" channel with a bot inside it:
server.setupChannel("test", {
    "name": "bob",
    "auxColor": "red"
});
  1. Create the "test" channel with a bot that creates the "abc" context and places a bot in it:
server.setupChannel("test", {
    "onCreate": `@
        create({
            auxCreator: null,
            auxContext: "abc",
        });
        create({
            auxCreator: null,
            auxLabel: "This is in the context!",
            auxColor: "blue",
            abc: true
        });
        destroy(this);
    `
});
