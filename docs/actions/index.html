Actions

#Data Actions

#create(parent, ...mods)

declare function create(parent: Bot | string, ...datas: Mod[]): Bot | Bot[];

Creates a new bot or combination of bots with the given #aux.creator and mods. Also triggers #onCreate() on all the created bots.

The first parameter is the bot or #id that should be used as the #aux.creator for the new bot(s). If you specify null, then the new bot(s) will not have a #aux.creator.

The other parameters are mods that should be applied to the new bot(s). If no other parameters are specified, then the new bot will have no data besides its #id. If an array of mods is used for a parameter, then one bot will be created for each unique combination of mods.

#Examples:

  1. Create an empty bot:
let myNewBot = create();
  1. Create a red bot:
let myRedBot = create(null, {
    "aux.color": "red"
});
  1. Create a parent and a child bot:
let myParentBot = create(null, {
    "aux.label": "Parent"
});
let myChildBot = create(myParentBot, {
    "aux.label": "Child"
});
  1. Create a bot from multiple mods:
// myBot is placed in the "myContext" context and is colored green
let myBot = create(null, mod.addToContext("myContext"), {
    "aux.color": "green"
});
  1. Create a red bot and a blue bot:
let [myRedBot, myBlueBot] = create(null, [
    {
        "aux.color": "red"
    },
    {
        "aux.color": "blue"
    }
]);

#destroy(bot)

declare function destroy(bot: Bot | string | Bot[]): void;

Removes the given bot, list of bots, or bot by #id and triggers #onDestroy() for each of the destroyed bots.

Note that only destroyable bots will be destroyed. Bots which have #aux.destroyable set to false will not be destroyed.

Also note that all bots which have #aux.creator set to the #id of a destroyed bot will also be destroyed (unless they are not destroyable). This happens recursively until there are no more bots to destroy.

#Examples:

  1. Destroying a single bot:
// Destroy a single bot that has #name set to "bob"
destroy(getBot("#name", "bob"));
  1. Destroying a list of bots:
// Destroy all the bots that have #aux.color set to "red"
destroy(getBots("#aux.color", "red"));
  1. Destroying a bot by its id
// Destroy the bot with the #id: "config"
destroy("config");

#Event Actions

#shout(name, arg?)

declare function shout(name: string, arg?: any): any[];

Sends a shout to all bots that are #aux.listening and have a listen tag for the specified name. Optionally includes a custom that argument. Also triggers #onListen() and #onAnyListen().

The first parameter is the name of the shout. e.g. Using "onClick" for the name will trigger all #onClick() listeners.

The second parameter is the that argument to send with the shout. You do not need to specify this parameter if you do not want to.

#Examples:

  1. Send a #reset() event to all bots:
shout("reset");
  1. Send a #hello() event with your name:
shout("hello", "Bob");

#whisper(bot, name, arg?)

declare function whisper(bot: (Bot | string)[] | Bot | string, eventName: string, arg?: any): any[];

Sends a whisper to the specified bot(s) that are #aux.listening and have a listen tag for the given name. Optionally includes a custom that argument. Also triggers #onListen() and #onAnyListen().

The first parameter is the bot, array of bots, bot #id, or array of bot #id that the whisper should be sent to.

The second parameter is the name of the whisper. e.g. Using "onClick" for the name will trigger the #onClick() listener for the specified bots.

The third parameter is the that argument to send with the shout. You do not need to specify this parameter if you do not want to.

#Examples:

  1. Send a #reset() event to all bots named "Bob":
let bots = getBots("#name", "Bob");
whisper(bots, "reset");
  1. Send a #setColor() event to ourself:
whisper(this, "setColor", "red");
